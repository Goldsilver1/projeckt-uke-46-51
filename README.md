
# Note Website

**Note Website** er et enkelt notatsystem utviklet som et skoleprosjekt.  
Målet med prosjektet er å lage en funksjonell webapplikasjon der brukeren kan opprette, redigere, lagre og slette notater lokalt via en Raspberry Pi 4B.

---

## Formål og beskrivelse

Prosjektet skal gjøre det mulig for brukeren å:
- Opprette nye notater  
- Redigere eksisterende notater  
- Slette notater  
- Lagre innholdet i en SQL-database på Raspberry Pi 4B

Applikasjonen kjøres lokalt på Raspberry Pi og er ment som en personlig notatplattform.  
Prototypen/MVP (Minimum Viable Product) består av et enkelt webgrensesnitt der brukeren kan skrive notater, som vist under utviklingen:

> Brukergrensesnittet består av:
> - Venstreside: liste over notater
> - Høyreside: skrivefelt for tittel og innhold

---

## Språk, teknologier og utstyr

| Type | Verktøy / Teknologi | Beskrivelse |
|------|----------------------|--------------|
| Frontend | HTML, CSS, JavaScript | For å lage selve nettsiden, grensesnittet og funksjonaliteten for å skrive/redigere notater |
| Backend / Database | SQL | Lagrer og henter notater fra databasen |
| Maskinvare | Raspberry Pi 4B, Laptop, Monitor, Tastatur, Mus, HDMI-kabel | Brukes for utvikling, testing og drift av systemet |

---

## Fagkompetanse

Dette prosjektet viser kompetanse i følgende fagområder:

### Utvikling
- Programmering av nettside med HTML, CSS og JavaScript  
- Bruk av SQL for lagring og henting av data  
- Testing av funksjoner (opprett, rediger, slett notat)

### Drift
- Oppsett av lokal webserver på Raspberry Pi 4B  
- Konfigurasjon av database og tilkobling  
- Forståelse av sikkerhet og vedlikehold i et lite websystem  

### Brukerstøtte
- Utforming av et enkelt og intuitivt brukergrensesnitt  
- Dokumentasjon av hvordan systemet brukes  
- Mulighet for å lage brukerveiledning og hjelpefunksjon
